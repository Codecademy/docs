The **`exp()`** function returns _e_ raised to the power of x.

## Syntax

```pseudo
exp($x)
```

- `$x`: Specifies the exponent that _e_ will be raised to.

## Codebyte Example

The following codebyte illustrates the use of the `exp()` function with a variety of arguments:

```codebyte/php
<?php
  echo (exp(0)."\n");
  echo (exp(1). "\n");
  echo (exp(2). "\n");
  echo (exp(4.8). "\n");
  echo (exp(10). "\n");
?>
```
