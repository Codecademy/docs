Reverse the elements in the list.

## Syntax

```py
list.reverse()
```

The `.reverse()` method does not take in any arguments.

## Example 1

```python
counting = [1, 2, 3, 4, 5]

counting.reverse()

print(counting)
# Output: [5, 4, 3, 2, 1]
```
