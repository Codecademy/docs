Defines a cell in a table that contains data.

## Syntax

```html
<td>Data</td>
```

**Note:** There are two kinds of cells, a header cell (contains header information) created by the `<th>` element and data cell(contains data) created by the `<td>` element.

## Example 1

Defining data cells in a table.

```html
<table>
  <tr>
    <th>Company</th>
    <th>Phone</th>
  </tr>
  <tr>
    <!-- Data cells -->
    <td>Apple</td>
    <td>iPhone</td>
  </tr>
  <tr>
    <!-- Data cells -->
    <td>Samsung</td>
    <td>Galaxy</td>
  </tr>
</table>
```

| Company | Phone  |
| ------- | ------ |
| Apple   | iPhone |
| Samsun  | Galaxy |
