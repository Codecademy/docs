Returns the inverse tangent of the argument in radians.

## Syntax

```java
Math.atan(n)
```

## Example 

Use `Math.atan()` to return the inverse tangent of `1.0`:

```java
public class Main {
  public static void main(String[] args) {
  
    double x = 1.0;
    
    System.out.println(Math.atan(x));
    // Output: 0.7853981633974483
  }
}
```
