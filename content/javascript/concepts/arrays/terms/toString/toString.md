## Definition

Returns a string with each of the array values, separated by commas. Does not mutate the original array.

## Syntax

```js
array.toString();
```

The `toString()` method does not have any parameters.

## Examples

```js
const numbers = [1, 2, 3];

console.log(numbers.toString());
// Output: '1,2,3'

console.log(numbers);
// Output: [1, 2, 3]
```
