## Definition

Remove existing whitespace from both ends of a string.

## Syntax

```js
string.trim()
```

## Example

Removing all existing whitespace from both ends of a string using `.trim()` method:

```js
console.log("  Hello World   ".trim());
// Output: Hello World
```
