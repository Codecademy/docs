The **.size** property returns the number of entries in a `Map` object.

## Syntax

```pseudo
set.size;
```

No parameters are needed for the `.size` property.

## Codebyte Example

```codebyte/javascript
const mySet = new Set();
mySet.add("text");
mySet.add(5);

console.log(mySet.size);
```
